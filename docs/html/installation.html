Installation¶

Prerequisites¶

  1. python >= 3.6

  2. pip

  3. CCAM(Center for Cell Analysis Modelling) account credentials to access HPC(UConn Health)

  4. Computer cluster server host(HPC UConn Node)

Run the following command to install from PyPi:

pip install biosimulations-dispatch
